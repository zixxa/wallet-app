Электронный кошелек

Приложение позволяет пользователю пересылать деньги на кошелек другому пользователю в различной валюте. Пользователь может иметь несколько кошельков прикрепленных к профилю. Операции перевода записываются в лист транзакций прикрепленных к кошельку.

По умолчанию деньги хранятся в рублях, платежу можно назначить валюту назначена в базе.


1. Инициализация базы данных

Запускается через Makefile в Docker, вся информация в конфигe docker-compose-local.yaml

Команды:
    Cоздать контейнер: make up
    Удалить контейнер: make down

2.Установка зависимостей

    cd config/requirements
    pip install -r base.txt

3. Миграции

    python manage.py makemigrations
    python manage.py migrate

4. Создание суперпользователя

    python manage.py createsuperuser

5. Запуск приложения

    python manage.py runserver

6. Запуск тестов

    python manage.py test wallet
